# 📊 Month 1 Project – Python Basics & Data Manipulation

## 📌 Overview

This project covers the fundamentals of Python programming and basic data manipulation techniques. The work is divided into four weeks, each focusing on essential concepts required for data analysis and visualization.

---

## 🎯 Objectives

* Learn Python programming basics
* Understand data structures and functions
* Perform data manipulation using NumPy and Pandas
* Visualize data using Matplotlib and Seaborn

---

## 📅 Weekly Breakdown

### 🔹 Week 1: Python Basics

* Learned variables, data types, operators
* Used conditional statements and loops
* Created simple programs (calculator, temperature converter)
* Built a script to calculate average temperature

---

### 🔹 Week 2: Data Structures & Functions

* Worked with lists, tuples, dictionaries, and sets
* Implemented functions, lambda expressions, and list comprehension
* Created a data cleaning script (removed duplicates, filtered data)

---

### 🔹 Week 3: NumPy & Pandas

* Used NumPy arrays for numerical operations
* Worked with Pandas DataFrames and Series
* Performed data cleaning and aggregation
* Calculated statistics like averages

---

### 🔹 Week 4: Data Visualization

* Created plots using Matplotlib
* Used Seaborn for advanced visualization
* Built a dashboard with scatter plots, histograms, and heatmaps

---

## 🛠 Tools & Technologies

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn

---

## 🚀 Key Outcomes

* Strong understanding of Python basics
* Ability to clean and process data
* Experience working with real datasets
* Skills in data visualization and analysis

---

## 📌 Conclusion

This project helped build a solid foundation in Python and data handling. The knowledge gained in this month is essential for advancing into data science and machine learning.
